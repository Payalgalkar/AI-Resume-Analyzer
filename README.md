# AI Resume Analyzer

An AI-powered Resume Analyzer built using **Flask** and **Google Gemini API**.

This application analyzes uploaded resumes, generates an ATS score, identifies strengths and weaknesses, recommends missing keywords, and provides career guidance through an AI assistant.

---

## Features

* Upload Resume (PDF)
* AI Powered Resume Analysis
* ATS Score Generation
* Resume Improvement Suggestions
* Missing Keyword Detection
* AI Career Assistant
* Responsive Dashboard

---

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* Google Gemini API
* PyPDF2

---

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Resume-Analyzer.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run the project

```bash
python app.py
```
